sports_scores.py grabs information from yahoo's sports page and gives the user
the current scores of the day.

**********
IMPORTANT
**********
In order to run this program, you will need the beautiful soup library. Works
with 3.4 version.

To use, run python sports_scores.py on terminal. You will be prompted to enter
in a sport. The sports that currently work are NBA, NHL, NFL, and MLB.

Live scores do not currently work. Will be added later.
